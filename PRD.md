# 项目需求文档：Markdown 笔记应用 (Markdown Note App)

## 1. 项目简介
实现一个功能完备的 Web 端 Markdown 笔记应用，支持即时创作、笔记管理以及美观的渲染输出。

---

## 2. 核心功能需求

### 📂 笔记管理
- **新建笔记**：用户可点击按钮创建新笔记，包含标题输入和内容编辑。
- **笔记列表**：左侧侧边栏展示所有笔记标题。
- **实时搜索**：支持在列表上方通过关键词快速检索笔记。
- **数据持久化**：使用 `LocalStorage` 进行本地存储，确保页面刷新或关闭后数据不丢失。
- **删除操作**：支持移除指定笔记，并同步更新存储。

### ✍️ 编辑与渲染
- **分栏布局**：
  - **左侧**：Markdown 源代码编辑器。
  - **右侧**：实时预览渲染后的 HTML 效果。
- **实时同步**：编辑器内容变动时，预览区须即时反馈。
- **代码高亮**：渲染出的代码块需支持语法高亮（如使用 Highlight.js 或 Prism.js）。

---

## 3. 界面结构 (UI Design)

| 模块 | 功能描述 |
| :--- | :--- |
| **侧边栏 (Sidebar)** | 搜索框、新建按钮、笔记标题列表 |
| **编辑器 (Editor)** | 文本输入区域，支持 Markdown 语法 |
| **预览区 (Preview)** | 渲染区域，展示格式化后的文档和高亮代码 |
